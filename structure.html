The basic anatomy of an HTML document. This is the header in an HTML document.

To specify it as a header, you have to utilize the heading level tags which range from h1 to h6.

Below, you will find instructions on how to structure a basic HTML document

  1. You must start every HTML document with "!DOCTYPE html" which is a required preamble that ensures your document behaves correctly.
  2. Second, use the "html" element. This element wraps all the content on the entire page and is sometimes known as the root element.
  3. Third, the "head" element acts as a container for all the information you want to include on an HTML page that is not the content you are showing.
  4. Following the "head" element, the "meta charset = utf-8" element sets your document to use UTF-8 which includes most characters from the majority of written documents. This element ensures that your document is fit to withstand the textual content you include on it.
  5. Fourthly, the "title" element sets the title for your page which appears in the brower tab that your page is loaded in.
  6. Lastly, the "body" element contains all the content that you want to show to web users when they visit your page, whether that's text, images, videos, games, playable audio tracks, or whatever else.

You can either use the shortcut "html" or an "!" to automatically make these elements appear in order on your HTML document.
