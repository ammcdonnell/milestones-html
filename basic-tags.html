Basic Tags

Block-level text containers

A block-level element always starts on a new line and takes up the full width available

A "p" tag, a "blockquote" tag, and a "pre" tag are all examples of block level HTML elements

A p-tag is used to define a simple paragraph of text

A "blockquote" element indicates that the enclosed text is an extended quotation. Here is an example of the "blockquote" element in action:

What makes a computer special? That it is a machine that can simulate another machine.

~ Jesse Farmer, Adjacent Academies

For the blockquote tag, a URL for the source of the quotation may be given using the cite attribute, and the "cite" element (used above), can give a text representation of the source.

The text in a "pre" element is displayed in a fixed-width font, usually in the Courier font.

The pre tag is often used to display language examples because it displays the code as it is typed.

Inline Text Styling

Contrast to block-level text containers, inline text elements do not start on a new line and only take up as much width as necessary.

The "em", "i", "strong", and "b" tags are all examples of inline text elements in HTML.

The "em" element adds an emphasis to a particular inline text. This elements takes on an italic form in the broswer:

The "i" tag is similar to the "em" tag as it defines a part of text in an alternate voice or mood which is typically displayed in italic

The i tag can be used to indicate technical terms, phrases in other languages, or a thought.

Lists

A lot of the web's content is lists and HTML has special elements for these. Marking up lists always consists of at least 2 elements. The most common list types are ordered and unordered lists:

  1. Unordered lists are for lists where the order of the items doesn't matter, such as a shopping list. These are wrapped in a "ul" element.
  2. Ordered lists are for lists where the order of the items does matter, such as a recipe. These are wrapped in an "ol" element.

Each item inside the lists is put inside an "li" (list item) element.
