Basic Tags

Block-level text containers

A block-level element always starts on a new line and takes up the full width available

A "p" tag, a "blockquote" tag, and a "pre" tag are all examples of block level HTML elements

A p-tag is used to define a simple paragraph of text

A "blockquote" element indicates that the enclosed text is an extended quotation. Here is an example of the "blockquote" element in action:

What makes a computer special? That it is a machine that can simulate another machine.

~ Jesse Farmer, Adjacent Academies

For the blockquote tag, a URL for the source of the quotation may be given using the cite attribute, and the "cite" element (used above), can give a text representation of the source.

The text in a "pre" element is displayed in a fixed-width font, usually in the Courier font.
The text in a "pre" element preserves    spaces  and
  line breaks. Thus,   the text is displayed exactly as it is
  written in the    HTML source code.

The pre tag is often used to display language examples because it displays the code as it is typed.
