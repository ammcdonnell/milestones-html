Nesting

  • In this code, the anchor tag is inside the paragraph tag, thus the anchor tag is a child tag of the paragraph tag. This means that if a change is made to the paragraph, the anchor “inherits” this change as well because it is inside the parent tag.

Relationship between opening and closing tags

  • An opening tag represents the beginning of an element and comprises of a less-than sign followed by an element’s name, and ends with a greater-than sign
  • A closing tagrepresents the end of an element and comprises of a less-than sign followed by a forward slash and the element’s name, and ends with a greater-than sign
  • For example, on the code above, the paragraph tag you can see the opening paragraph tag at the beginning of the code and the closing paragraph tag at the end of the code

Anatomy of an Anchor tag

  • The content in between the opening and closing tags is the material of that element. In between the opening and closing tags of an anchor link is its content which consists of attributes.
  • Attributes are properties that are used to provide additional information about an element. In this snippet, there are four attributes.
  • Attributes include a name and a value and are defined within the opening tag, after an element's name
  • For example, in the snippet above, the value of the href attribute is "https://google.com". The value of the id attribute is “the best link”, the value of the class attribute is “banana”, and the value of the target attribute is "_blank">Google

Purpose and Effect of anchor tag attributes (as seen in code below)

  • The id attribute identifies an element
  • The class attribute classifies an element
  • The src attribute specifies a source for embeddable content
  • The href attribute provides a hyperlink reference to a linked resource
  • The target attribute distinguishes where to display the linked URL as the name for a browsing context (the current tab, a new window, etc.). The target tag has keywords that have special meanings for where to load the URL. For example the _self keyword is the default and specifies to load the URL in the current browsing context. The _blank tag, which is in the above snippet, specifies to load the URL in a new tab.
